Netflix Data Quality Enhancement and Preprocessing using Python

Project Objective:

The objective of this project is to clean and preprocess the Netflix Movies and TV Shows dataset by handling missing values, checking duplicates, standardizing formats, validating data quality, and preparing the dataset for analysis.

Dataset Information:

The dataset contains information about Netflix Movies and TV Shows including title, director, cast, country, release year, rating, duration, and description.

Data Cleaning Steps Performed:
1. Dataset inspection
2. Missing value analysis
3. Missing value treatment
4. Duplicate record analysis
5. Column name standardization
6. Text standardization
7. Date format conversion
8. Data validation checks
9. Data quality reporting

Tools Used:
1. Python
2. Pandas
3. Jupyter Notebook

Final Results:
Total Records: 8807
Total Columns: 12
Missing Values Remaining: 0
Duplicate Records: 0
Invalid Release Years: 0

Output Files:
cleaned_netflix_titles.csv
cleaning_report.txt

Conclusion:

The dataset was successfully cleaned and preprocessed. Data quality issues were identified and resolved, making the dataset ready for further analysis and visualization.
